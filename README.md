# Apex Tech - 全方位網頁開發與 AI 智慧解決方案

Apex Tech 專業提供 RWD 響應式網頁、SEO 優化與 AI 智慧系統整合，協助企業在數位轉型浪潮中保持領先。

## 預覽專案

以下提供兩個不同風格的專案版本，點擊即可在新頁面瀏覽：

### 🎨 經典風格版
[**開啟 index.html**](index.html) — 乾淨簡潔的設計，以紅橙漸層為主色調，適合追求簡約高效的團隊。

### 🔥 頂尖風格版
[**開啟 apex-modern.html**](apex-modern.html) — 充滿活力的日落漸層搭配毛玻璃效果，適合追求視覺衝擊的現代化場景。
