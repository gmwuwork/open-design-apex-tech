# Apex Tech - 全方位網頁開發與 AI 智慧解決方案

Apex Tech 專業提供 RWD 響應式網頁、SEO 優化與 AI 智慧系統整合，協助企業在數位轉型浪潮中保持領先。

## 預覽專案

以下提供兩個不同風格的專案版本，點擊即可在瀏覽器新分頁中瀏覽：

### 🎨 經典風格版
<a href="https://gmwuwork.github.io/open-design-apex-tech/" target="_blank" rel="noopener noreferrer"><strong>開啟 index.html</strong></a><br/>
<span style="color: #6b7280; font-size: 0.9em;">乾淨簡潔的設計，以紅橙漸層為主色調，適合追求簡約高效的團隊。</span>

### 🔥 頂尖風格版
<a href="https://gmwuwork.github.io/open-design-apex-tech/apex-modern.html" target="_blank" rel="noopener noreferrer"><strong>開啟 apex-modern.html</strong></a><br/>
<span style="color: #6b7280; font-size: 0.9em;">充滿活力的日落漸層搭配毛玻璃效果，適合追求視覺衝擊的現代化場景。</span>
